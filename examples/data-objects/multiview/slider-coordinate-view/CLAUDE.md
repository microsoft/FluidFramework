# @fluid-example/multiview-slider-coordinate-view

Slider view component for editing coordinates in the multiview sample.

## Build

- `pnpm build` - Build the package
- `pnpm build:esnext` - TypeScript compilation only
- `pnpm build:copy` - Copy CSS files to lib
- `pnpm lint` - Run linting
- `pnpm lint:fix` - Fix lint issues

## Notes

- This is an example package (private, not published)
- Part of the multiview example demonstrating separation of model and view
- React-based view component with slider controls for coordinate editing
- Depends on `@fluid-example/multiview-coordinate-interface`
- Library-only package (no runnable app)
