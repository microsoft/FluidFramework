## `flub rename-types`

Renames type declaration files from .d.ts to .d.mts.

```
USAGE
  $ flub rename-types [-v | --quiet]

LOGGING FLAGS
  -v, --verbose  Enable verbose logging.
      --quiet    Disable all logging.

DESCRIPTION
  Renames type declaration files from .d.ts to .d.mts.
```
