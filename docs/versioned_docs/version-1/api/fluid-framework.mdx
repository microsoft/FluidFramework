# fluid-framework Package

The fluid-framework package bundles a collection of Fluid Framework client libraries for easy use when paired with a corresponding service client package (for example, [@fluidframework/azure-client](../deployment/azure-fluid-relay.mdx) or [@fluidframework/tinylicious-client](../testing/tinylicious.mdx)).

The `fluid-framework` packages surfaces APIs from the following sub-packages.
The following APIs can be referenced independently, but they can all be imported directly from the `fluid-framework` package:

-   [@fluidframework/map](./map/index.md)
-   [@fluidframework/sequence](./sequence/index.md)
-   [@fluidframework/fluid-static](./fluid-static/index.md)
-   [@fluidframework/container-definitions](./container-definitions/index.md)
