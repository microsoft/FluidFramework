# Copilot and Claude Slash Commands

Claude Code and Github Copilot CLI have different but similar built-in slash-commands.
If you are coming to Copilot from Claude, this mapping might be helpful.
Note that the commands are similar but not the same in all cases.

## Claude Code --> Github Copilot CLI

- `/btw` -> `/ask`
- `/simplify` -> `/review` — not as effective in my experience, but similar in intent
- Plan mode -> `/plan <type your plan>`
- `/insights` -> `/chronicle`
