Provides an in memory implementation of [ITokenProvider](/docs/apis/routerlicious-driver/itokenprovider-interface) that can be used to insecurely connect to the Fluid Relay.

As the name implies, this is not secure and should not be used in production. It simply makes examples where authentication is not relevant easier to bootstrap.
