## Method Details

### compare (ALPHA) {#compare-methodsignature}

Return a number representing the ascending sort order of the `a` and `b` code details:

- `< 0` if `a < b`.

- `= 0` if `a === b`.

- `> 0` if `a > b`.

- `undefined` if `a` is not comparable to `b`.

Similar semantics to: [Array.sort](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort#description)

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.
