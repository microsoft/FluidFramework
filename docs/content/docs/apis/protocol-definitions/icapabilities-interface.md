## Property Details

### interactive {#interactive-propertysignature}

Indicates if the client represents a potentially interactive session with a user (if 'true') or if it's a "system entity" (if 'false').

#### Signature {#interactive-signature}

```typescript
interactive: boolean;
```

#### Remarks {#interactive-remarks}

The only "system entity" scenario at the moment is the [summarizer client](https://fluidframework.com/docs/concepts/summarizer/).
