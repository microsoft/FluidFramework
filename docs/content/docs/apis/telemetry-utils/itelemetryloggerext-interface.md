## Signature {#itelemetryloggerext-signature}

```typescript
export interface ITelemetryLoggerExt extends ITelemetryBaseLogger
```

**Extends:** [ITelemetryBaseLogger](/docs/apis/core-interfaces/itelemetrybaselogger-interface)

## Remarks {#itelemetryloggerext-remarks}

This interface is meant to be used internally within the Fluid Framework, and `ITelemetryBaseLogger` should be used when loggers are passed between layers.
