This library contains a simple React-based visualizer for the Fluid Devtools.

## Remarks {#_fluid-experimental_devtools-view-remarks}

The entry-point to this package is [DevtoolsPanel](/docs/apis/devtools-view#devtoolspanel-function), a [React Component](https://react.dev/reference/react/Component) for displaying debug information, which can be added to your Fluid-backed React app.
