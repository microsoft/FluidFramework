Provides a hierarchical organization of map-like data structures as SubDirectories. The values stored within can be accessed like a map, and the hierarchy can be navigated using path syntax. SubDirectories can be retrieved for use as working directories.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

## Signature {#shareddirectory-signature}

```typescript
/** @sealed */
export declare class SharedDirectory extends SharedObject<ISharedDirectoryEvents> implements ISharedDirectory
```

**Extends:** [SharedObject](/docs/apis/shared-object-base/sharedobject-class)&lt;[ISharedDirectoryEvents](/docs/apis/fluid-framework/ishareddirectoryevents-interface)&gt;

**Implements:** [ISharedDirectory](/docs/apis/fluid-framework/ishareddirectory-interface)

## Example {#shareddirectory-example}

```typescript
mySharedDirectory.createSubDirectory("a").createSubDirectory("b").createSubDirectory("c").set("foo", val1);
const mySubDir = mySharedDirectory.getWorkingDirectory("/a/b/c");
mySubDir.get("foo"); // returns val1
```
