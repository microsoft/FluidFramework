### takeNextCreationRange (ALPHA) {#takenextcreationrange-methodsignature}

Returns a range of IDs created by this session in a format for sending to the server for finalizing. The range will include all IDs generated via calls to `generateCompressedId` since the last time this method was called.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#takenextcreationrange-signature}

```typescript
takeNextCreationRange(): IdCreationRange;
```

#### Returns {#takenextcreationrange-returns}

the range of IDs, which may be empty. This range must be sent to the server for ordering before it is finalized. Ranges must be sent to the server in the order that they are taken via calls to this method.

**Return type:** [IdCreationRange](/docs/apis/id-compressor/idcreationrange-interface)
