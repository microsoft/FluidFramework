## Property Details

### electedClientId (ALPHA) {#electedclientid-propertysignature}

Most recently elected client id. This is either:

1. the interactive elected parent client, in which case electedClientId === electedParentId, and the SummaryManager on the elected client will spawn a summarizer client, or

2. the non-interactive summarizer client itself.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#electedclientid-signature}

```typescript
readonly electedClientId: string | undefined;
```

### electedParentId (ALPHA) {#electedparentid-propertysignature}
