## Property Details

### disableGC (ALPHA) {#disablegc-propertysignature}

Flag that if true, will disable garbage collection for the session. Can be used to disable running GC on containers where it is allowed via the gcAllowed option.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#disablegc-signature}

```typescript
disableGC?: boolean;
```

### enableGCSweep (ALPHA) {#enablegcsweep-propertysignature}

Flag that if true, will enable the full Sweep Phase of garbage collection for this session, where Tombstoned objects are permanently deleted from the container.

IMPORTANT: This only applies if this document is allowed to run Sweep Phase.

Current default behavior is for Sweep Phase not to delete Tombstoned objects, but merely to prevent them from being loaded.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#enablegcsweep-signature}

```typescript
enableGCSweep?: true;
```

### gcAllowed (ALPHA) {#gcallowed-propertysignature}

Flag that if true, will enable running garbage collection (GC) for a new container.

GC has mark phase and sweep phase. In mark phase, unreferenced objects are identified and marked as such in the summary. This option enables the mark phase. In sweep phase, unreferenced objects are eventually deleted from the container if they meet certain conditions. Sweep phase can be enabled using the "enableGCSweep" option.

Note: This setting is persisted in the container's summary and cannot be changed.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#gcallowed-signature}

```typescript
gcAllowed?: boolean;
```

### runFullGC (ALPHA) {#runfullgc-propertysignature}

Flag that will bypass optimizations and generate GC data for all nodes irrespective of whether a node changed or not.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#runfullgc-signature}

```typescript
runFullGC?: boolean;
```

### sessionExpiryTimeoutMs (ALPHA) {#sessionexpirytimeoutms-propertysignature}

Maximum session duration for a new container. If not present, a default value will be used.

Note: This setting is persisted in the container's summary and cannot be changed.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#sessionexpirytimeoutms-signature}

```typescript
sessionExpiryTimeoutMs?: number;
```

### sweepGracePeriodMs (ALPHA) {#sweepgraceperiodms-propertysignature}

Delay between when Tombstone should run and when the object should be deleted. This grace period gives a chance to intervene to recover if needed, before Sweep deletes the object. If not present, a default (non-zero) value will be used.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#sweepgraceperiodms-signature}

```typescript
sweepGracePeriodMs?: number;
```
