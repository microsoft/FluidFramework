### initialContainers {#initialcontainers-propertysignature}

(optional) List of Containers to initialize the devtools with.

#### Signature {#initialcontainers-signature}

```typescript
initialContainers?: ContainerDevtoolsProps[];
```

#### Remarks {#initialcontainers-remarks}

Additional Containers can be registered with the Devtools via [registerContainerDevtools](/docs/apis/devtools/idevtools-interface#registercontainerdevtools-methodsignature).

### logger {#logger-propertysignature}

(optional) telemetry logger associated with the Fluid runtime.

#### Signature {#logger-signature}

```typescript
logger?: DevtoolsLogger;
```

#### Remarks {#logger-remarks}

Note: the Devtools do not register this logger with the Fluid runtime; that must be done separately.

This is provided to the Devtools instance strictly to enable communicating supported / desired functionality with external listeners.
