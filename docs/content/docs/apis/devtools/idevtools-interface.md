Fluid Devtools. A single, global instance is used to generate and communicate stats associated with the general Fluid runtime (i.e., it is not associated with any single Framework entity).

## Signature {#idevtools-signature}

```typescript
export interface IDevtools extends IDisposable
```

**Extends:** IDisposable

## Remarks {#idevtools-remarks}

Supports registering [IFluidContainer](/docs/apis/fluid-static/ifluidcontainer-interface)s for Container-level stats (via [registerContainerDevtools](/docs/apis/devtools/idevtools-interface#registercontainerdevtools-methodsignature)).

The lifetime of the associated singleton is bound by that of the Window (globalThis), and it will be automatically disposed of on Window unload. If you wish to dispose of it earlier, you may call its _@fluidframework/core-interfaces\#IDisposable.dispose_ method.
