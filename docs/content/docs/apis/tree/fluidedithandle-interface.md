FluidEditHandles are used to load edit chunks stored outside of the EditLog. Can be satisfied by IFluidHandle<ArrayBufferLike>. Note that though this is in `PersistedTypes`, it isn't directly serializable (e.g. `get` is a function). Its serialization relies on being encoded via an IFluidSerializer.
