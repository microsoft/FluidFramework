Plain text.

## Signature {#plaintextnode-signature}

```typescript
export declare class PlainTextNode implements DocumentationLiteralNode<string>, SingleLineDocumentationNode
```

**Implements:** [DocumentationLiteralNode](/docs/apis/api-markdown-documenter/documentationliteralnode-interface)&lt;string&gt;, [SingleLineDocumentationNode](/docs/apis/api-markdown-documenter/singlelinedocumentationnode-interface)

## Remarks {#plaintextnode-remarks}

Must not contain any line breaks.

To include line breaks in your text, use [LineBreakNode](/docs/apis/api-markdown-documenter/linebreaknode-class) in a container node like [SpanNode](/docs/apis/api-markdown-documenter/spannode-class) or [ParagraphNode](/docs/apis/api-markdown-documenter/paragraphnode-class).
