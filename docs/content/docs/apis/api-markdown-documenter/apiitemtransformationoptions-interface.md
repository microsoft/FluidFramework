## Property Details

### createChildContentSections {#createchildcontentsections-propertysignature}

Shared transformation logic for generating child content sections within a section describing an API item that potentially has child elements (see [TransformApiItemWithChildren](/docs/apis/api-markdown-documenter#transformapiitemwithchildren-typealias)).

#### Signature {#createchildcontentsections-signature}

```typescript
createChildContentSections?: CreateChildContentSections;
```

#### Remarks {#createchildcontentsections-remarks}

This method is used by the default transformation implementatios. This can be used to adjust the layout of the child sections for API item kinds that have without having to provide new transformation overrides for all of those content types.
