Represents errors associated with network communication.

## Signature {#networkerror-signature}

```typescript
export declare class NetworkError extends Error
```

**Extends:** Error

## Remarks {#networkerror-remarks}

The Fluid Framework server implementation includes a collection of services that communicate with each other over the network. Network communication is subject to a diverse range of errors. [NetworkError](/docs/apis/server-services-client/networkerror-class) helps convey more information than a simple HTTP status code, allowing services to be aware of the context of a network error and making those services more prepared to react to such kinds of errors.
