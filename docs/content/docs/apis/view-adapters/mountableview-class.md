[Packages](/docs/apis/) &gt; [@fluidframework/view-adapters](/docs/apis/view-adapters) &gt; [MountableView](/docs/apis/view-adapters/mountableview-class)

Abstracts mounting of views for usage outside of their bundle. Supports React elements.

The MountableView must be applied from within the same bundle that provides the view, and then that MountableView can be used by a separate bundle. Attempting to apply a MountableView to a view that was retrieved from a separate bundle is not supported.

## Signature {#mountableview-signature}

```typescript
export declare class MountableView implements IFluidMountableView
```

**Implements:** [IFluidMountableView](/docs/apis/view-interfaces/ifluidmountableview-interface)
