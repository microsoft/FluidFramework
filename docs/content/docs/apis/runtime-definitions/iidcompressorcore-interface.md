### takeNextCreationRange {#takenextcreationrange-methodsignature}

Returns a range of local IDs created by this session in a format for sending to the server for finalizing. The range will include all local IDs generated via calls to `generateCompressedId` since the last time this method was called.

#### Signature {#takenextcreationrange-signature}

```typescript
takeNextCreationRange(): IdCreationRange;
```

#### Returns {#takenextcreationrange-returns}

the range of session-local IDs, which may be empty. This range must be sent to the server for ordering before it is finalized. Ranges must be sent to the server in the order that they are taken via calls to this method.

**Return type:** [IdCreationRange](/docs/apis/runtime-definitions/idcreationrange-interface)
