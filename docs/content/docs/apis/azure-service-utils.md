A set of helper utilities for building backend APIs for use with [Azure Fluid Relay](https://docs.microsoft.com/en-us/azure/azure-fluid-relay/overview/overview).

Note that this library's primary entry-point ([generateToken](/docs/apis/azure-service-utils#generatetoken-function)) is only intended to be run in a browser context. It is \*\*not\*\* Node.js-compatible.

## Function Details

### generateToken {#generatetoken-function}

Generates a [JSON Web Token](https://en.wikipedia.org/wiki/JSON_Web_Token) (JWT) to authorize access to a Routerlicious-based Fluid service.

#### Signature {#generatetoken-signature}

```typescript
export declare function generateToken(tenantId: string, key: string, scopes: ScopeType[], documentId?: string, user?: IUser, lifetime?: number, ver?: string): string;
```

#### Remarks {#generatetoken-remarks}

Note: this function uses a browser friendly auth library ([jsrsasign](https://www.npmjs.com/package/jsrsasign)) and may only be used in client (browser) context. It is \*\*not\*\* Node.js-compatible.
