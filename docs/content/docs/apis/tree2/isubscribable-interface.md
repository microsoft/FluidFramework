An object which allows the registration of listeners so that subscribers can be notified when an event happens.

[createEmitter(noListeners)](/docs/apis/tree2#createemitter-function) can help implement this interface via delegation. `EventEmitter` can be used as a base class to implement this via extension.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.
