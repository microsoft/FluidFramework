## Property Details

### checkout (ALPHA) {#checkout-propertysignature}

Access non-view schema specific aspects of of this branch.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#checkout-signature}

```typescript
readonly checkout: ITreeCheckout;
```

#### Remarks {#checkout-remarks}

This is a non-owning reference: disposing of this view does not impact the branch.

### context (ALPHA) {#context-propertysignature}

Context for controlling the EditableTree nodes produced from [editableTree](/docs/apis/tree2/flextreeview-interface#editabletree-propertysignature).

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#context-signature}

```typescript
readonly context: FlexTreeContext;
```

#### Remarks {#context-remarks}

This is an owning reference: disposing of this view disposes its context.

### editableTree (ALPHA) {#editabletree-propertysignature}

```typescript
readonly editableTree: FlexTreeTypedField<TRoot>;
```

## Method Details

### fork (ALPHA) {#fork-methodsignature}

Spawn a new view which is based off of the current state of this view. Any mutations of the new view will not apply to this view until the new view is merged back into this view via `merge()`.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#fork-signature}

```typescript
fork(): ITreeViewFork<TRoot>;
```

**Return type:** [ITreeViewFork](/docs/apis/tree2/itreeviewfork-interface)&lt;TRoot&gt;
