## Property Details

### schema (ALPHA) {#schema-propertysignature}

The schema stored in the document.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#schema-signature}

```typescript
readonly schema: TreeStoredSchema;
```

#### Remarks {#schema-remarks}

Edits to the schema can mutate the schema stored of the tree which took this snapshot (but this snapshot will remain the same) This is mainly useful for debugging cases where schematize reports an incompatible view schema.
