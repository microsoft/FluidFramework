## Property Details

### attach (ALPHA) {#attach-propertysignature}

When set, indicates that some content is being attached from the given detached field.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#attach-signature}

```typescript
readonly attach?: DetachedNodeId;
```

### count (ALPHA) {#count-propertysignature}

The number of nodes affected. When `isAttachMark(mark)` is true, this is the number of new nodes being attached. When `isAttachMark(mark)` is false, this the number of existing nodes affected. Must be 1 when `fields` is populated.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#count-signature}

```typescript
readonly count: number;
```

### detach (ALPHA) {#detach-propertysignature}

When set, indicates that some pre-existing content is being detached and sent to the given detached field.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#detach-signature}

```typescript
readonly detach?: DetachedNodeId;
```

### fields (ALPHA) {#fields-propertysignature}

Modifications to the pre-existing content. Must be undefined when `attach` is set but `detach` is not.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#fields-signature}

```typescript
readonly fields?: FieldMap<TTree>;
```
