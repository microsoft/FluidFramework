### is (BETA) {#is-methodsignature}

Narrow the type of the given value if it satisfies the given schema.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#is-signature}

```typescript
is<TSchema extends TreeNodeSchema>(value: unknown, schema: TSchema): value is NodeFromSchema<TSchema>;
```

#### Example {#is-example}

```ts
if (node.is(myNode, point)) {
    const y = myNode.y; // `myNode` is now known to satisfy the `point` schema and therefore has a `y` coordinate.
}
```

### key (BETA) {#key-methodsignature}

The key of the given node under its parent.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#key-signature}

```typescript
key(node: NodeBase): string | number;
```

#### Remarks {#key-remarks}

If `node` is an element in a [TreeArrayNode](/docs/apis/tree2/treearraynode-interface), this returns the index of `node` in the list (a `number`). Otherwise, this returns the key of the field that it is under (a `string`).
