## Type Details

### ArrayToUnion (BETA) {#arraytounion-typealias}

Convert a Array type into the type of ReadonlySet.

Same as `keyof ListToKeys<T, unknown>` but work for values that are not valid keys.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#arraytounion-signature}

```typescript
export type ArrayToUnion<T extends readonly unknown[]> = T extends readonly (infer TValue)[] ? TValue : never;
```

### ExtractItemType (BETA) {#extractitemtype-typealias}

Get the `Item` type from a `LazyItem<Item>`.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#extractitemtype-signature}

```typescript
export type ExtractItemType<Item extends LazyItem> = Item extends () => infer Result ? Result : Item;
```

### FlexList (BETA) {#flexlist-typealias}

A flexible way to list values. Each item in the list can either be an "eager" \*\*value\*\* or a "lazy" \*\*function that returns a value\*\* (the latter allows cyclic references to work).

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#flexlist-signature}

```typescript
export type FlexList<Item = unknown> = readonly LazyItem<Item>[];
```

### FlexListToUnion (BETA) {#flexlisttounion-typealias}

Normalize FlexList type to a union.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#flexlisttounion-signature}

```typescript
export type FlexListToUnion<TList extends FlexList> = ExtractItemType<ArrayToUnion<TList>>;
```

### LazyItem (BETA) {#lazyitem-typealias}

An "eager" or "lazy" Item in a `FlexList`. Lazy items are wrapped in a function to allow referring to themselves before they are declared. This makes recursive and co-recursive items possible.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#lazyitem-signature}

```typescript
export type LazyItem<Item = unknown> = Item | (() => Item);
```
