Provides a means for interacting with a SharedTree. This includes reading data from the tree and running transactions to mutate the tree.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

## Signature {#itreecheckout-signature}

```typescript
export interface ITreeCheckout extends AnchorLocator
```

**Extends:** [AnchorLocator](/docs/apis/tree2/anchorlocator-interface)

## Remarks {#itreecheckout-remarks}

This interface should not have any implementations other than those provided by the SharedTree package libraries.

## Method Details

### fork (ALPHA) {#fork-methodsignature}

Spawn a new view which is based off of the current state of this view. Any mutations of the new view will not apply to this view until the new view is merged back into this view via `merge()`.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#fork-signature}

```typescript
fork(): ITreeCheckoutFork;
```

#### Returns {#fork-returns}

**Return type:** [ITreeCheckoutFork](/docs/apis/tree2/itreecheckoutfork-interface)

### merge (ALPHA) {#merge-methodsignature}

Apply all the new changes on the given view to this view.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#merge-signature}

```typescript
merge(view: ITreeCheckoutFork): void;
```

#### Remarks {#merge-remarks}
