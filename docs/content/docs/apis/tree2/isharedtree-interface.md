## Method Details

### contentSnapshot (ALPHA) {#contentsnapshot-methodsignature}

Provides a copy of the current content of the tree. This can be useful for inspecting the tree when no suitable view schema is available. This is only intended for use in testing and exceptional code paths: it is not performant.

This does not include everything that is included in a tree summary, since information about how to merge future edits is omitted.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#contentsnapshot-signature}

```typescript
contentSnapshot(): SharedTreeContentSnapshot;
```

#### Returns {#contentsnapshot-returns}

**Return type:** [SharedTreeContentSnapshot](/docs/apis/tree2/sharedtreecontentsnapshot-interface)
