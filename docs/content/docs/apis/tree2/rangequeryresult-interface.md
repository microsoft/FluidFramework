## Property Details

### length (ALPHA) {#length-propertysignature}

The length of the prefix of the query range which have the same value. For example, if a RangeMap has the same value for keys 5, 6, and 7, a query about the range \[5, 10\] would give a result with length 3.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.
