### set (BETA) {#set-methodsignature}

Adds or updates an entry in the map with a specified `key` and a `value`.

WARNING: This API is provided as a beta preview and may change without notice. Use at your own risk.

#### Signature {#set-signature}

```typescript
set(key: string, value: TIn | undefined): void;
```

#### Remarks {#set-remarks}

Setting the value at a key to `undefined` is equivalent to calling [delete(key)](/docs/apis/tree2/treemapnodebase-interface#delete-methodsignature) with that key.
