A collection of utilities for managing [StableNodeKey](/docs/apis/tree2#stablenodekey-typealias)s. A node key can be assigned to a node and allows that node to be easily retrieved from the tree at a later time. (see `nodeKey.map`).

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

## Signature {#nodekeys-signature}

```typescript
export interface NodeKeys
```

## Remarks {#nodekeys-remarks}

[LocalNodeKey](/docs/apis/tree2/localnodekey-interface)s are put on [FlexTreeObjectNode](/docs/apis/tree2/flextreeobjectnode-interface)s via a special field. A node with a node key in its schema must always have a node key.
