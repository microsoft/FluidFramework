### getAll (ALPHA) {#getall-methodsignature}

Retrieve all key/offset pairs stored on this segment. Entries should be ordered by offset, such that the `i`th result's attribution key applies to offsets in the open range between the `i`th offset and the `i+1`th offset. The last entry's key applies to the open interval from the last entry's offset to this collection's length.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.

#### Signature {#getall-signature}

```typescript
getAll(): IAttributionCollectionSpec<T>;
```

#### Returns {#getall-returns}

**Return type:** [IAttributionCollectionSpec](/docs/apis/merge-tree/iattributioncollectionspec-interface)&lt;T&gt;
