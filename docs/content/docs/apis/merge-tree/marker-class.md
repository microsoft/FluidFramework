Markers are a special kind of segment that do not hold any content.

Markers with a reference type of [Tile](/docs/apis/merge-tree/referencetype-enum#tile-enummember) support spatially accelerated queries for finding the next marker to the left or right of it in sub-linear time. This is useful, for example, in the case of jumping from the start of a paragraph to the end, assuming a paragraph is bound by markers at the start and end.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.
