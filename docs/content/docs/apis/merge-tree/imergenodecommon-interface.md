### ordinal (ALPHA) {#ordinal-propertysignature}

A string that can be used for comparing the location of this node to other `MergeNode`s in the same tree. `a.ordinal < b.ordinal` if and only if `a` comes before `b` in a pre-order traversal of the tree.
