## Property Details

### attach (ALPHA) {#attach-propertysignature}

Enables tracking attribution information for operations on this merge-tree. This function is expected to subscribe to appropriate change events in order to manage any attribution data it stores on segments.

This must be done in an eventually consistent fashion.

WARNING: This API is provided as an alpha preview and may change without notice. Use at your own risk.
