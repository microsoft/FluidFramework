## Property Details

### aggregateBlobsSmallerThanBytes {#aggregateblobssmallerthanbytes-propertysignature}

Give hosts the option to change blob aggregation behavior to suit their needs. Larger number means fewer blob individual requests, but less blob-deduping. Smaller number means more blob individual requests, but more blob-deduping. Setting to `undefined` disables blob aggregration. Default: undefined

#### Signature {#aggregateblobssmallerthanbytes-signature}

```typescript
aggregateBlobsSmallerThanBytes: number | undefined;
```

### enableDiscovery {#enablediscovery-propertysignature}

```typescript
enableDiscovery?: boolean;
```

### enableInternalSummaryCaching {#enableinternalsummarycaching-propertysignature}

Enable internal cache of summaries/snapshots. Reduces Summarizer boot time and reduces server load in E2E tests. Default: true

#### Signature {#enableinternalsummarycaching-signature}

```typescript
enableInternalSummaryCaching: boolean;
```

### enableLongPollingDowngrade {#enablelongpollingdowngrade-propertysignature}
