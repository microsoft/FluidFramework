ITelemetryLogger interface contains various helper telemetry methods, encoding in one place schemas for various types of Fluid telemetry events. Creates sub-logger that appends properties to all events

{{% callout warning This API is deprecated and will be removed in a future release. %}}
For internal use within FluidFramework, use ITelemetryLoggerExt in \@fluidframework/telemetry-utils. No replacement intended for FluidFramework consumers.

{{% /callout %}}
