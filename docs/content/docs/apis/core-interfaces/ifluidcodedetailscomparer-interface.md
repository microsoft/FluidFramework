### compare {#compare-methodsignature}

Return a number representing the ascending sort order of the `a` and `b` code details; `< 0` if `a < b`. `= 0` if `a === b`. `> 0` if `a > b`. `undefined` if `a` is not comparable to `b`.

Similar semantics to: [Array.sort](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort#Description)
