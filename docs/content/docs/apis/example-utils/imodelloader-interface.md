### createDetached {#createdetached-methodsignature}

Create a detached model using the specified version of container code. Returns an object containing the detached model plus an attach callback. When invoked, the attach callback returns a promise that will resolve after attach has completed with the id of the container.

#### Signature {#createdetached-signature}

