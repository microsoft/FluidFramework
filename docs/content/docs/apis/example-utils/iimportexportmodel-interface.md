## Property Details

### exportData {#exportdata-propertysignature}

Export the data from the model. Can be passed into importData() for a new container to replicate the data.

#### Signature {#exportdata-signature}

```typescript
exportData: () => Promise<ExportType>;
```

### importData {#importdata-propertysignature}

importData must be called after initialization but before modifying or attaching the model (i.e. can only be called on an unaltered, detached model).

#### Signature {#importdata-signature}

```typescript
importData: (initialData: ImportType) => Promise<void>;
```

### supportsDataFormat {#supportsdataformat-propertysignature}

Permit format checking in a generic manner - without knowing the type of our data or the type of the model, we can still check whether the model supports that data.

#### Signature {#supportsdataformat-signature}

```typescript
supportsDataFormat: (initialData: unknown) => initialData is ImportType;
```
