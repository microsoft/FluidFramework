# fluidframework.com

Our public website can be found at <https://fluidframework.com/>.

## Website Contents

The website is made up of a mix of authored and generated content.

Authored contents can be found under [docs/content](https://github.com/microsoft/FluidFramework/tree/main/docs/content).
For guidelines and tips for authoring website content, see the [Markdown authoring guidance](https://github.com/microsoft/FluidFramework/tree/main/docs#writing-markdown-content).

The website also publishes automatically generated API documentation for our public library packages.
This documentation can be found in the [API documentation](https://fluidframework.com/docs/apis/).
These contents are generated using [API-Extractor](https://api-extractor.com/) and our [@fluid-tools/api-markdown-documenter](https://github.com/microsoft/FluidFramework/blob/main/tools/api-markdown-documenter/README.md) library.
For more details on our API documentation generation, see the [API documentation generation notes](https://github.com/microsoft/FluidFramework/tree/main/docs#api-documentation).

## Website Build

Documentation regarding building / testing the website locally can be found in the [docs README](https://github.com/microsoft/FluidFramework/tree/main/docs#readme).
