# fluidframework.com

Our public website can be found at <https://fluidframework.com/>.

## Website Contents

The website is made up of a mix of authored and generated content.

Authored contents can be found under the [`website`](../../../website) directory.
For guidelines and tips for authoring website content, see the [website authoring guidance](../../../website/README.md#writing-site-documentation).

The website also publishes automatically generated API documentation for our public library packages.
This documentation can be found in the [API documentation](https://fluidframework.com/docs/apis/).
These contents are generated using [API-Extractor](https://api-extractor.com/) and our [@fluid-tools/api-markdown-documenter](../../../tools/api-markdown-documenter/README.md) library.
For more details on our API documentation generation, see the [website API docs build notes](../../../website/README.md#local-api-docs-build).

Documentation regarding building / testing the website locally can be found in the [website README](../../../website/README.md).
