### apiModel {#apimodel-propertysignature}

#### Signature {#apimodel-signature}

```typescript
apiModel: ApiModel;
```

#### Remarks {#apimodel-remarks}

Beyond being the root entry for rendering, this is used to resolve member links globally, etc.

If you need to generate a model from API reports on disk, see [loadModel()](/docs/apis/api-markdown-documenter#loadmodel-function)<!-- -->.

### logger {#logger-propertysignature}

Policy object for logging system events.

#### Signature {#logger-signature}

```typescript
readonly logger?: Logger;
```

#### Remarks {#logger-remarks}

A custom logger can be provided for customized policy, or for a target other than the console.

If you wish to enable `verbose` logging, consider using [verboseConsoleLogger](/docs/apis/api-markdown-documenter#verboseconsolelogger-variable)<!-- -->.
