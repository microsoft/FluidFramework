## Remarks {#linebreaknode-remarks}

Note that [PlainTextNode](/docs/apis/api-markdown-documenter\plaintextnode-class) does not support line breaks.

To build up a grouping of text including line breaks, use this type alongside text nodes within a container type like [ParagraphNode](/docs/apis/api-markdown-documenter\paragraphnode-class) or [SpanNode](/docs/apis/api-markdown-documenter\spannode-class)<!-- -->.
