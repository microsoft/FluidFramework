Events that trigger when the roster of members in the Fluid session change. Only changes that would be reflected in the returned map of [IServiceAudience](/docs/apis/fluid-static/iserviceaudience)<!-- -->'s [IServiceAudience.getMembers()](/docs/apis/fluid-static/iserviceaudience#getmembers-MethodSignature) method will emit events.

<b>Signature:</b>

```typescript
export interface IServiceAudienceEvents<M extends IMember> extends IEvent 
```
<b>Extends:</b> [IEvent](/docs/apis/common-definitions/ievent)

## Remarks

The following is the list of events emitted.

### "membersChanged"

The "membersChanged" event is emitted when a member is either added or removed.

#### Listener signature

```typescript
() => void;
```
### "memberAdded"

The "memberAdded" event is emitted when a member joins the audience.

#### Listener signature

```typescript
(clientId: string, member: M) => void;
```
- `clientId` - A unique identifier for the client

- `member` - The service-specific member object for the client

### "memberRemoved"

The "memberRemoved" event is emitted when a member leaves the audience.

#### Listener signature

```typescript
(clientId: string, member: M) => void;
```
- `clientId` - A unique identifier for the client

- `member` - The service-specific member object for the client
