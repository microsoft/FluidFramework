[Packages](/docs/apis/) &gt; [@fluidframework/sequence](/docs/apis/sequence) &gt; [IntervalCollection](/docs/apis/sequence\intervalcollection-class)

Collection of intervals that supports addition, modification, removal, and efficient spatial querying. This class is not a DDS in its own right, but emits events on mutating operations such that it's possible to integrate into a DDS. This aligns with its usage in `SharedSegmentSequence`<!-- -->, which allows associating intervals to positions in the sequence DDS which are broadcast to all other clients in an eventually consistent fashion.
