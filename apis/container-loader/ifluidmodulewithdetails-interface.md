### details {#details-propertysignature}

Code details associated with the module. Represents a document schema this module supports. If the code loader implements the <i>@fluidframework/core-interfaces\#IFluidCodeDetailsComparer</i> interface, it'll be called to determine whether the module code details satisfy the new code proposal in the quorum.
