## Property Details

### childCount {#childcount-propertysignature}

Number of direct children of this node

#### Signature {#childcount-signature}

```typescript
childCount: number;
```

### children {#children-propertysignature}

Array of child nodes.

#### Signature {#children-signature}

```typescript
children: IMergeNode[];
```

#### Remarks {#children-remarks}

To avoid reallocation, this is always initialized to have maximum length as deemed by the merge tree's branching factor. Use `childCount` to determine how many children this node actually has.

### partialLengths {#partiallengths-propertysignature}

Supports querying the total length of all descendants of this IMergeBlock from the perspective of any (clientId, seq) within the collab window.

#### Signature {#partiallengths-signature}

```typescript
partialLengths?: PartialSequenceLengths;
```

#### Remarks {#partiallengths-remarks}

This is only optional for implementation reasons (internal nodes can be created/moved without immediately initializing the partial lengths). Aside from mid-update on tree operations, these lengths objects are always defined.
