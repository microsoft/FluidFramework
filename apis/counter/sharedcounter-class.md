A shared object that holds a number that can be incremented or decremented.

## Signature {#sharedcounter-signature}

```typescript
export declare class SharedCounter extends SharedObject<ISharedCounterEvents> implements ISharedCounter 
```
<b>Extends:</b> [SharedObject](/docs/apis/shared-object-base\sharedobject-class)<!-- -->&lt;[ISharedCounterEvents](/docs/apis/counter\isharedcounterevents-interface)

<b>Implements:</b> [ISharedCounter](/docs/apis/counter\isharedcounter-interface)

## Remarks {#sharedcounter-remarks}

Note that `SharedCounter` only operates on integer values. This is validated at runtime.

## Examples {#sharedcounter-examples}

### Example 1 {#sharedcounter-example1}

Creating a `SharedCounter`<!-- -->:

First, get the factory and call [IChannelFactory.create()](/docs/apis/datastore-definitions\ichannelfactory-interface#create-methodsignature) with a runtime and string ID:

```typescript
const factory = SharedCounter.getFactory();
const counter = factory.create(this.runtime, id) as SharedCounter;
```
The initial value of a new `SharedCounter` is 0. If you wish to initialize the counter to a different value, you may call [SharedCounter.increment()](/docs/apis/counter\sharedcounter-class#increment-method) before attaching the Container, or before inserting it into an existing shared object.

### Example 2 {#sharedcounter-example2}

Using the `SharedCounter`<!-- -->:

Once created, you can call [SharedCounter.increment()](/docs/apis/counter\sharedcounter-class#increment-method) to modify the value with either a positive or negative number:

```typescript
counter.increment(10); // add 10 to the counter value
counter.increment(-5); // subtract 5 from the counter value
```
To observe changes to the value (including those from remote clients), register for the [incremented](/docs/apis/counter\isharedcounterevents-interface) event:

```typescript
counter.on("incremented", (incrementAmount, newValue) => {
    console.log(`The counter incremented by ${incrementAmount} and now has a value of ${newValue}`);
});
```
