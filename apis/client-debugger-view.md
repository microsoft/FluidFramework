[Packages](/docs/apis/) &gt; [@fluid-tools/client-debugger-view](/docs/apis/client-debugger-view)

Contains an extensible debug visualizer / editor for the Fluid client.

## Remarks {#_fluid-tools_client-debugger-view-remarks}

This package has 2 primary entry-points:

- [renderClientDebuggerView()](/docs/apis/client-debugger-view#renderclientdebuggerview-function)<!-- -->: A general-purpose rendering utility for displaying debug information.

- <i>FluidClientDebugger</i>: A [React Component](https://reactjs.org/docs/react-component.html) for displaying debug information, which can be added to your Fluid-backed React app.
