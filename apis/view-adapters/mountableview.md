Abstracts mounting of views for usage outside of their bundle. Supports React elements, as well as objects that implement IFluidHTMLView.

The MountableView must be applied from within the same bundle that provides the view, and then that MountableView can be used by a separate bundle. Attempting to apply a MountableView to a view that was retrieved from a separate bundle is not supported.

<b>Signature:</b>

```typescript
export declare class MountableView implements IFluidMountableView 
```
<b>Implements:</b> [IFluidMountableView](/docs/apis/view-interfaces/ifluidmountableview)
