### containerNickname {#containernickname-propertysignature}

Optional: Nickname for [the Container](/docs/apis/client-debugger\fluidclientdebuggerprops-interface#container-propertysignature) / debugger instance.

#### Signature {#containernickname-signature}

```typescript
containerNickname?: string;
```

#### Remarks {#containernickname-remarks}

Associated tooling may take advantage of this to differentiate between debugger instances using semantically meaningful information.

If not provided, the [FluidClientDebuggerProps.containerId](/docs/apis/client-debugger\fluidclientdebuggerprops-interface#containerid-propertysignature) will be used for the purpose of distinguising debugger instances.
